# D6 Dice Roller

A simple D6 dice roller built with Python in Jupyter Notebook.

## Features
- Roll a single D6 dice with visual display
- Roll multiple dice at once
- Track roll statistics (total, average, highest, lowest)
- View distribution chart of all rolls
- Clear history to start fresh

## How to Use
1. Open `dice_roller.ipynb` in Jupyter Notebook
2. Run the first cell to load the functions
3. Run "Roll a Single Dice" or "Roll Multiple Dice" cells
4. Check your statistics anytime with "View Statistics"
